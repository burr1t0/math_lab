# Лабораторная работа по математическому анализу  
**Тема:** Логистическое отображение и точечные динамические системы  

- Рассмотрено понятие **точечного отображения** и его связь с динамическими системами.  
- Изучено **логистическое отображение** как модель роста популяции с ограниченными ресурсами.  
- Проведены доказательства базовых свойств: ограниченность последовательности, наличие неподвижных точек, условия монотонности.  
- Построены графики зависимости \( x_{n+1} \) от \( x_n \) для различных значений параметра \( r \).  
- Исследованы подпоследовательности, пределы и устойчивость неподвижных точек.  
- Рассмотрены циклы порядка \( m \) и метод **лестницы Ламерея** для графического анализа сходимости.  

## Общий вывод
Логистическое отображение является фундаментальной моделью дискретной динамики.  
- При малых значениях параметра \( r \) последовательность стремится к нулю.  
- При \( r \in (1;3] \) наблюдается устойчивое стремление к неподвижной точке.
